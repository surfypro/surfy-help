# Nouveautés (version alpha)

Cette page liste les **évolutions présentes sur la branche de développement / alpha** et **pas encore publiées** dans le journal « production » ([Nouveautés](./app.md)). Elle sert de **brouillon** : vous pouvez y noter plus de détails, des formulations provisoires ou des regroupements à simplifier avant la mise en prod.

Lors d'une **mise en production**, le contenu pertinent est **reformulé si besoin**, puis **déplacé ou recopié** dans `app.md` sous la version et la date de release ; les entrées alpha correspondantes peuvent être retirées ou réduites ici.

Les règles de style (public utilisateur, pas de nom de client, balises `<P />`, `<LIV />`, etc.) sont les mêmes que pour la production — voir le [README du dossier changelog](./README).
