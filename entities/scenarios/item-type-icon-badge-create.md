# Créer un type d'objet avec badge sur l'icône

Lors de la création d’un <OT code="itemType" />, associez d’abord une **icône**, puis un **badge** (<P code="itemType:iconBadge" />) : bande verticale à droite **dans** le carré de l’icône, avec texte court et couleurs. Utile pour distinguer des variantes qui partagent le même pictogramme (ex. extincteurs CO² / EAU / ABC).

## Prérequis

- Droits de modification sur les types d’objets.
- Connaître le pictogramme à utiliser (ex. extincteur).

## Étapes

1. **Ouvrez la création d’un type d’objet**  
   Point de départ : fiche de création d’un <OT code="itemType" />.

2. **Saisissez le Nom du type d'objet**  
   Exemple : *Extincteur CO²*.

3. **Changer la famille de types d'objet**  
   Ouvre la liste des familles pour choisir celle du type.

4. **Choisissez Sécurité incendie**  
   Sélectionnez la famille dans la liste.

5. **Changer d'icone**  
   Le badge n’est disponible **que** lorsqu’une icône est déjà définie. Choisissez le pictogramme extincteur.

6. **Ouvre l'éditeur de badge (texte et couleurs)**  
   Cliquez sur **Créer un badge** pour ouvrir l’éditeur.

7. **Saisissez le Texte du badge**  
   Exemple : *CO²* (caractère Unicode **²**, pas un « 2 » classique).

8. **Saisissez la couleur de fond**  
   Ouvrez la palette et sélectionnez une couleur pour la bande.

9. **Saisissez la couleur du texte**  
   Ouvrez la palette et sélectionnez une couleur pour le texte du badge.

10. **Cliquez sur le bouton Valider la création**  
    Enregistre le type d’objet avec l’icône et le badge.

## Limites

- Sans icône sur le type, le badge reste indisponible.
- Le texte du badge est court ; le rendu est une bande dans le carré d’icône (pas à côté).
