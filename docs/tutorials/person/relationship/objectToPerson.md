# Association d'objet à personne

## Créer une association d'objet à personne

On peut associer un objet à une personne dans Surfy, tel qu'un casier, un siège ergonomique ou un caisson.

### Type d'objets à associer

Le type d'objets concerné doit avoir l'information "Est associable à une personne ?" cochée.

Pour cela,

-   cliquer sur un des objets concernés sur le plan
-   dans la fiche de l'objet à gauche du plan, cliquer sur l'icône en début de ligne du "Type d'objet" pour "Voir plus d'option pour le type d'objet XXX" et pour "Modifier le type d'objet XXX"
-   cocher la case "Est associable à une personne ?"
-   sauvegarder les modifications.

Tous les objets issus de ce type d'objet pourront être associés à une personne.

Pour associer un objet à une personne,

-   cliquer sur l'objet à associer
-   cliquer sur "associer une personne à cet objet" dans la fiche de l'objet à gauche du plan
-   rechercher et sélectionner la personne à associer
-   La personne est associée à l'objet.

En cliquant sur l'objet dans le plan, le nom de la personne associée apparaît dans sa fiche.<br />
La fiche de la personne associée liste les objets qui lui sont associés.<br />
La recherche de la personne ou de l'objet avec la loupe informe de l'association.
En cliquant sur l'objet, on revient sur le plan à l'objet désigné.<br />

### Visualiser les objets associables libres

On peut visualiser la disponibilité des objets associables sur les plans.<br />
Cela permet une gestion efficace des casiers libres, par exemple.

Pour visualiser les objets associables libres, depuis le plan,

-   cliquer sur l'icône verticale "GPS" pour "Colorier les postes de travail par rapport à leur état"
-   cliquer sur "Colorier les objets sans association"

Les objets associables libres se colorient en vert sur le plan.

## Supprimer une association d'objet à personne

-   cliquer sur l'objet à dissocier
-   cliquer sur l'icône "Poubelle" face au nom de la personne associée pour "Supprimer l'association d'objet à personne XXX-YYY"
-   confirmer la suppression d'association
-   la relation d'objet à personne est supprimée
