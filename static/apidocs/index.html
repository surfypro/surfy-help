get the jwt token (open id)

get the jwt token using api credentials created inside surfy using open id style

Authorizations:
BearerAuth
Request Body schema: application/json
required
client_id
string

the client id is the tenant

client_secret
string

the client secret can be found or created inside the api section of surfy app

Responses

Default api server

https://app.surfy.pro/api/v1/authentication/openapi/oauth/token

Request samples

  • Payload
Content type
application/json
{
  • "client_id": "demo",
  • "client_secret": "mypassword"
}

Response samples

  • 200
Content type
application/json
{
  • "ok": true,
  • "token": "string",
  • "access_token": "string",
  • "expires_in": 0
}
