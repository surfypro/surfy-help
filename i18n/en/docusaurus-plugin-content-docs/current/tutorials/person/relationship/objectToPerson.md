# Object to person association



<Youtube code="fZhxgYYguHY"/>

## Create an object to person association

You can associate an object with a person in Surfy, such as a locker, an ergonomic chair, or a cabinet.

### Object types to associate

The relevant object type must have the "Can be associated with a person?" option checked.

To do this,

-   click on one of the relevant objects on the plan
-   in the object card to the left of the plan, click on the icon at the beginning of the "Object type" row to "See more options for object type XXX" and "Modify object type XXX"
-   check the "Can be associated with a person?" box
-   save the modifications.

All objects from this object type can be associated with a person.

To associate an object with a person,

-   click on the object to associate
-   click on "associate a person to this object" in the object card to the left of the plan
-   search for and select the person to associate
-   The person is associated with the object.

By clicking on the object in the plan, the name of the associated person appears in its card.<br />
The associated person's card lists the objects associated with them.<br />
Searching for the person or object with the magnifying glass shows the association.
By clicking on the object, you return to the plan at the designated object.<br />

### View unassigned associable objects

You can view the availability of associable objects on the plans.<br />
This allows efficient management of available lockers, for example.

To view unassigned associable objects, from the plan,

-   click on the vertical "GPS" icon to "Color workstations by their status"
-   click on "Color objects without association"

Unassigned associable objects are colored green on the plan.

## Delete an object to person association

-   click on the object to dissociate
-   click on the "Trash" icon next to the associated person's name to "Delete object to person association XXX-YYY"
-   confirm association deletion
-   the object to person relationship is deleted
