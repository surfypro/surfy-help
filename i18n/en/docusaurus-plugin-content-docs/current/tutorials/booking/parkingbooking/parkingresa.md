# Access to parking booking

## Introduction

Booking in Surfy is accessible to collaborators in the application via a browser, or from Teams by adding the "Surfy" application.

To access Surfy via a browser, go to [app.surfy.pro](https://app.surfy.pro/login) from Chrome, Edge, Firefox, Safari,...<br />
The collaborator then uses their account to access the features they are authorized for.

To add the Surfy application in Teams, click on "Applications" and search for "Surfy" to add it to Teams applications.
Once added in Teams, the application provides access to the authorized Surfy features.

Both access methods lead to the same features.<br />
Using Teams offers a more direct experience and is better suited for using booking features on a smartphone.

## Book a parking space

Parking space booking uses the same interface as [workstation booking](/en/docs/tutorials/booking/workstationbooking/workplace).

The user whose associated collaborator is correctly configured (see the [parking booking getting started guide](/en/docs/tutorials/booking/parkingbooking/admin)) can then book a parking space in buildings showing the letter "P" in the summary of their assignments, at the top of the booking page.

By clicking on "BOOK", the "Book a parking space" box allows you to choose the appropriate parking space on the plan for the desired day. Available spaces are colored green.<br />
Clicking on a parking space on the plan confirms the booking for the chosen day.<br />
Clicking on the trash icon releases the parking space booking.

Half-day booking is possible and can be made mandatory, particularly for electric vehicles.
